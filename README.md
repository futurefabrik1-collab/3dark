# 3DARK — Wir verwandeln reale Räume in digitale Assets

Website for 3DARK, specialists in photorealistic Digital Twins and interactive 3D environments using Gaussian Splatting.

**Production URL**: https://3dark.de

## Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn UI components

## Local Development

```sh
git clone <YOUR_GIT_URL>
cd 3dark
npm i
npm run dev
```
